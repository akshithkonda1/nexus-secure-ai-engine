# Toron v2.5H+ Testing Ecosystem Skeleton

This tree mirrors the DevOps control panel layout used by the FastAPI and React implementation.

- `backend/`: API wiring, orchestration hooks, and persistence contracts.
- `frontend/`: UI wiring, component registry, and page blueprints for the Vite app.
- `sim/`: deterministic simulation cases.
- `telemetry/`: scrubbing/quarantine policies.
- `schemas/`: shared data contracts.
- `ci_cd/`: placeholders for pipelines and quality gates.
- `docs/`: control-plane runbooks and diagrams.
